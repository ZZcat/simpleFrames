Quadratic

Quadratics are a type of function like a linear or constant function. We call the shape of linear and constant lines to be a straight line. We call the shape of a quadratic a parabola. On the left is an image of a constant, linear and quadratic function.

Color Type Example
Red Constant y=1
Blue Linear y=0.5x+1
Green Quadratic y=x^2+x-2

Features of a Quadratic

A quadratic function has a lot more features than a linear function. It is a curve that changes direction once. A curve is simpily a function who's slope (a.k.a. rate of change) changes over time. If we look at the equation of a linear line in slope-intercept form, y = mx + b, we'll see that the slope of the line is always m.
